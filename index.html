Online Wallpapers

A simple to use Android wallpaper manager, that utlizes the reddit API to fetch the latest wallpapers shared by the platform members, And gives users the ability to choose from endless varity of wallpapers. Users are given a group of categories to select the type of wallpapers they want to be fetched to them. As well as the abilty to browse through them before setting one. Also it has the ability to automaticlly set a new random wallpaper picked from the chosen categories with given duration set by the user too.

Nevertheless, This project is an open-source project and is based on open-source technologies ReactJS and React Native. This is the first beta release of the app, so bugs and errors are expected to occur frequently in this stage of development, So please do use it with that in mind.
